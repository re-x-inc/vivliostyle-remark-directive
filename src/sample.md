# シンプルなサンプル

これは最小限のremark-directiveデモです。

:::div{.note}
これは注記です。青色の枠で表示されます。
:::

:::div{.warning}
これは警告です。オレンジ色の枠で表示されます。
:::

## 基本的な使い方

通常のMarkdownテキストの中に、`:::div{.class}`構文でコンテナを埋め込むことができます。
